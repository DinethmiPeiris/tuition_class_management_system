Tuition Class Management System
Project ID: ISP_G21
Institution: Sri Lanka Institute of Information Technology (SLIIT)
Course: Information Systems Project 2026

Overview
The Tuition Class Management System is a comprehensive web-based platform designed to digitize and streamline the management of tuition classes. It replaces traditional manual processes (paper-based registration, attendance, payments, and communication) with an efficient, organized, and user-friendly digital solution.
This system serves as a complete academic and administrative management tool for tuition teachers and students, significantly reducing administrative workload, minimizing errors, and improving communication.

Key Features
1. Student Management

Online student registration with approval workflow
Automatic Student ID generation upon approval
Student status management (Active, Inactive, Suspended)
Registration status tracking

2. Course & Enrollment Management

Course and batch creation (by grade, subject & batch year)
Student enrollment requests with teacher approval
Monthly subject material upload and access control
Payment slip upload and verification system

3. Class & Attendance Management

Class timetable creation with conflict detection
Digital attendance marking
Monthly attendance summaries
Student timetable and attendance viewing

4. Examination & Academic Performance Management

Online exam scheduling and conduction
Digital exam submission and marking
Performance report generation
Result publishing and viewing

5. Academic Communication & Activity Management

Announcements management
In-system messaging between teachers and students
Activity logging and audit trail
Message history

Technology Stack

Frontend: sHTML, CSS, JavaScript
Backend: Java, Spring Boot
Database: SQL Server Management Studio
Authentication: JWT (JSON Web Tokens)
ORM: Hibernate (JPA)
Architecture: RESTful APIs, Three-tier Architecture
